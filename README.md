## Getting Started

1.
```
npm install
```

2.
```
npm install -g bower
```

**[Optional]** Permission errors? Try this:

```
sudo chown -R whoami /usr/local
```

3.
```
bower install
```

4.
```
npm install -g grunt-cli
```

5.
```
sudo gem install compass
```

6.
```
grunt
```

7.
```
grunt serve
```

8.
Visit [http://localhost:8000/](http://localhost:8000/)
